<h2>🛠️ Installation Steps:</h2>

<p>1. Clone</p>

```
git clone https://github.com/rohankarn35/AskMeAI.git
```

<p>2. Install dependencies</p>

```
flutter pub get
```

<p>3. Get your api key from OpenAI</p>

<p>4. Run the app in debug mode</p>

```
flutter run
```
